1. fetch origin
2. pull main
3. switch ไปที่ branch ของตัวเอง

        4.1.1 rebase จบแล้ว
    4.2 ถ้ามี-conflict
        4.2.1 แก้ไข conflict
        4.2.2 add conflict
        4.2.3 rebase --continue

6. merge branch ของตัวเอง เข้า main
7. push main
